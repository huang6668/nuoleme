## 使用提醒

- 首次安装后请确认短信权限、通知权限和全屏提醒状态
- 建议先点击一次“测试报警”，确认当前手机的闹铃、震动和锁屏提醒行为
- 如果你的系统管得比较严，记得把自启动、后台活动和电池优化放开

## 获取方式

当前默认通过 GitHub Actions 分发调试包：

1. 打开仓库的 Actions 页面
2. 进入 `Android Build`
3. 下载 `nuolemo-debug-apk`
4. 安装 `app-debug.apk`

## 已知边界

- App 不联网，不上传短信内容
- 不依赖常驻后台或轮询保活
- 不会自动修改你的媒体音量

## 反馈建议

如果你在某个具体机型上遇到“短信到了但提醒不明显 / 没有全屏 / 息屏漏报”，反馈时尽量带上：

- 手机品牌与系统版本
- 是否开启通知、全屏提醒、自启动、后台活动
- 短信示例是否命中你配置的关键词或车牌
