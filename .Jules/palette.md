## 2024-05-24 - Screen Reader Noise Reduction\n**Learning:** Decorative icons placed next to descriptive text shouldn't have redundant contentDescriptions, as screen readers will read both. Marking decorative ImageViews with `android:importantForAccessibility="no"` significantly improves screen reader clarity and flow by removing duplicate information.\n**Action:** When inspecting Android XML layouts, check if an `ImageView` is purely decorative and accompanied by text. If so, remove `contentDescription` and set `importantForAccessibility="no"` to reduce screen reader noise.
